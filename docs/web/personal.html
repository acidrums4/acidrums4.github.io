Personal

Propuesta gráfica para el rediseño del sitio web de la Dirección Nacional de Personal Académico y Administrativo de la Universidad Nacional de Colombia. Además de la página inicial, consta de subsitios web para las cuatro divisiones de la dirección, junto con sistemas de información e interfaces para la consulta de convocatorias y documentos.

Para efectos de la presentación en este portafolio, se excluyeron los elementos pertenecientes a la plantilla web institucional de la universidad, los cuales estaban incluídos en los bocetos originales.

Ver este sitio web

Por peticion de la dirección, varios elementos del diseño y la diagramación de la propuesta seleccionada (presentada en este portafolio) fueron alterados durante el proceso de maquetación. Actualmente, este sitio web se encuentra en desarrollo.
